# 投资决策框架 V1 (Investment Decision Framework)
投资决策框架,帮助你从情绪化决策转向系统化决策.

## 核心设计理念

### 传统投资 vs 系统化投资
**决策依据**, 传统投资=感觉,直觉,市场情绪, 系统化投资=数据,指标,规则
**研究时间**, 传统投资=每天 3-6 小时手忙脚乱, 系统化投资=30 分钟系统化检查
**回撤控制**, 传统投资=追涨杀跌,情绪化止损, 系统化投资=预警机制,系统性减仓
**学习积累**, 传统投资=复盘依赖记忆, 系统化投资=自动记录,持续优化

## 四层决策模型
```
第一层:流动性监控 (宏观环境)
 ↓
第二层:情绪监控 (市场温度)
第三层:价值评估 (标的选择)
第四层:抄底模型 (入场时机)

### 层级说明
**第一层:流动性**, 作用=判断宏观环境,是否适合持仓, 核心指标=净流动性,SOFR,MOVE 指数, 决策权重=40%
**第二层:情绪**, 作用=判断市场过热/过冷, 核心指标=NAAIM,散户资金流向,机构仓位, 决策权重=30%
**第三层:价值**, 作用=评估标的的合理性, 核心指标=ROE,负债率,现金流等, 决策权重=20%
**第四层:抄底**, 作用=捕捉极端行情机会, 核心指标=MVRV,恐慌指数,矿机关机价, 决策权重=10%

### Skill 3: 价值评估框架 (Value Assessment)
**作用**:评估标的的基本面合理性

**美股评估标准**:
判断标准:
 - ROE > 15%(持续 3 年以上)
 - 负债率 < 50%
 - 自由现金流 > 净利润的 80%
 - 护城河评估(品牌/网络效应/成本优势)

输出:
 - 投资评级: (A/B/C/D)
 - 理由: 关键支持数据

**加密货币评估标准**:
 - 市值排名: Top 20
 - 30 天交易量: > 稳定
 - 开发活跃度: GitHub commits/活跃度
 - 生态项目: 数量与质量
 - 持有者结构: 长期持有者占比 > 40%

 - 投资评级: (强烈推荐/推荐/中性/不推荐)

### Skill 4: 比特币抄底模型 (Bitcoin Bottom-Fishing)
**作用**:捕捉极端行情的抄底机会

 - RSI < 30(周线级别超跌), 交易量:恐慌抛售后成交量萎缩(<30 均量), MVRV 比率:< 1.0(市值低于实现市值)
 - 社交媒体恐慌指数: Twitter/Reddit > 75
 - 矿机关机价:现价接近 S19 Pro 成本线
 - 长期持有者行为:LTH 占比上升

 - 满足 4 个以上指标 → 分批建仓信号
 - 满足 5 个以上指标 → 重仓抄底信号

 - 抄底评级: (强/中/弱), 建议仓位比例: (10%/20%/30%/50%), 入场策略: (分批/一次性)
